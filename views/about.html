Personal Heatmap was built by Mehan Jayasuriya using NodeJS, Express, heatmap.js, HTML5, Canvas, JavaScript, Node Oauth, the Google Maps JavaScript API and the Open Paths API. Inspired, in part, by Where Do You Go.

Personal Heatmap was developed in John Schimmel's Spring 2012 Dynamic Web class at NYU ITP. Many thanks to John for all of his help with the project.

If you'd like to view or fork the source code of this project, head over to GitHub.
